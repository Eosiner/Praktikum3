# Perhitungan Kalkulator
Ini adalah projek sederhana kalkulator

# Cara pemakaian 
1. Pastikan memiliki compiler java pada perangkat anda
2. clone projek ini pada perangkat and
3. buka filenya 
4. kemudian run dan lihat outputnya
